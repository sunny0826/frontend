# 平台用户注册服务协议

依据《中华人民共和国民法典》《中华人民共和国个人信息保护法》等相关法律法规，甲乙双方就乙方注册使用甲方平台相关事宜，本着平等自愿、公平诚信的原则，协商一致签订本协议，以资共同恪守履行。

## 1. 协议生效与使用范围
1. 本协议自乙方完成甲方平台注册流程、根据自身主体类型勾选用户类别并点击确认同意本协议之日起生效，成为约束甲乙双方平台使用权利义务的有效法律文件。
2. 本协议适用于乙方在甲方平台的所有注册、使用行为，乙方通过平台进行的任何操作，均视为已充分理解并同意本协议全部条款；境外用户确认知晓并遵守甲方所在地（中国）相关法律法规及乙方所属地跨境数据、税收相关规则，因乙方所属地规则差异产生的法律责任由乙方自行承担。
3. 本协议中针对境内外个人用户的差异化约定，均为不可分割部分，对应类型乙方受相关条款专属约束。

## 2. 平台注册与账户使用
1. 乙方确认，根据自身真实主体类型选择用户类别，若信息发生变更，应及时在平台完成更新，否则由此产生的一切不利后果由乙方自行承担。
2. 乙方应妥善保管平台账户及登录密码、验证码等身份验证信息，对以其账户进行的所有操作及产生的结果承担全部责任，因账户泄露、盗用等非甲方原因造成的损失，甲方不承担赔偿责任。
3. 乙方承诺，其注册及使用平台账户的行为均符合法律法规及平台规则，不得利用账户从事违法、违规、违反公序良俗的活动，否则甲方有权暂停、冻结或注销乙方账户，并追究其法律责任。境外用户如因违法行为违反中国法律，甲方将配合中国司法机关处理，并按规定向相关部门报送信息。
4. 乙方不得擅自变更主体类型，确需变更的，应向甲方提出申请及证明材料，经甲方审核通过后方可变更。

## 3. 积分规则
1. 甲方根据乙方在开源平台的贡献情况核算并发放积分，积分分为可提现积分与不可提现积分两类，具体核算标准、积分类型划分由甲方在平台公示，甲方有权根据平台运营需要合理调整，调整后将通过平台公告方式通知乙方。

### a. 可提现积分
可提现积分按照乙方主体类型及所属地实行差异化规则，具体如下：
（1）境内个人用户：可按平台公示比例申请提现，确认提现具体信息后与甲方指定的第三方平台签约提现，具体以平台公示规则和第三方签约结果为准；
（2）境外个人用户：可按平台公示的外汇兑换比例（以甲方合作银行当日汇率为准）申请提现，需提供本人名下合法境外收款账户，甲方不代扣代缴相关税费，境外用户应自行遵守所属地税收规则，完成税务申报及缴纳，因未合规交税所产生的责任由乙方自行承担，提现涉及的外汇结算费用由乙方承担，具体标准以平台公示为准。

### b. 不可提现积分
仅可在甲方积分商城内兑换电子权益、代金券等平台指定权益，不得申请现金提现，积分兑换后不予退换，不可提现积分的有效期及使用规则以平台公示为准。

4. 乙方不得通过作弊、虚假提交、恶意刷取等任何违规方式获取积分，一经发现，甲方有权立即撤销乙方违规获取的全部积分，暂停或终止乙方积分使用权益，情节严重的，甲方可注销乙方平台账户。

## 4. 信息采集与使用
### a. 基础信息采集
为完成平台注册、积分提现、权益兑换等核心服务，乙方同意甲方采集其在注册及使用平台过程中主动提供的信息，包括但不限于姓名/名称、身份信息、联系地址、联系电话、电子邮箱、银行账户信息等。

### b. 开源平台信息采集与贡献核算
乙方自愿授权甲方采集其在GitHub、Gitee等开源平台公开账户的相关信息（包括但不限于代码提交记录、项目参与记录、Issue/PR互动记录、仓库信息等），甲方有权基于前述采集的信息，按照平台公示规则对乙方的开源贡献进行核算，并根据核算结果发放对应积分，乙方确认该授权为真实意思表示，且对其提供的开源平台账户信息享有合法权利。

### c. 信息使用范围
甲方采集的乙方信息，仅用于为乙方提供平台注册、积分核算、积分提现、权益兑换等约定服务，以及提升平台服务质量、优化平台功能的必要场景，未经乙方书面同意，不得用于本协议约定之外的其他用途（法律法规另有规定的除外）。

### d. 信息加密存储
甲方承诺对采集的乙方信息采取符合中国法律法规及行业标准的加密存储、访问控制等安全保护措施，建立完善的信息安全管理制度，配备专业的安全技术人员，防止信息泄露、篡改、丢失，保障乙方信息安全。

## 5. 账户信息分析与数据使用授权
1. 乙方同意并授权甲方对其平台账户信息、前述开源平台账户信息进行合规分析、数据整合及脱敏处理，分析过程严格遵循中国有关信息保护的法律法规。
2. 乙方授权甲方将经过脱敏、匿名化处理后的聚合数据/匿名数据（该数据无法单独或结合其他信息关联至乙方个人身份，不构成法律意义上的个人信息）进行使用，包括但不限于用于甲方平台的运营分析、开源生态研究、行业分析等合法用途。
3. 乙方同意甲方将前款所述的聚合数据/匿名数据向有合法需求的合作客户提供、转让或出售，该等行为不构成对乙方个人隐私信息的泄露，甲方无需另行取得乙方同意，合作客户应按照与甲方的约定合法使用该数据。
4. 甲方保证对未完成脱敏、匿名化处理的乙方账户信息及个人信息，不向任何第三方提供、转让或出售（法律法规要求披露或乙方书面同意的除外）。

## 6. 双方权利与义务
### 6.1 甲方权利与义务
1. 有权按照本协议约定及平台公示规则采集、分析乙方相关信息，核算并发放积分，对乙方的违规行为采取相应的处理措施。
2. 保障乙方正常使用平台服务及积分权益，维护积分提现、商城兑换的合规性与稳定性，及时响应乙方的合理咨询与投诉；为境外用户提供跨境提现的技术支持，协调合作银行完成外汇结算。
3. 严格履行乙方个人信息及账户信息的保密义务和安全保护义务，采取有效措施防止信息泄露，若因甲方过错导致乙方个人信息泄露并造成损失的，甲方应依法承担赔偿责任。
4. 有权根据法律法规及平台运营需要，对本协议条款、积分规则、平台服务内容等进行合理调整，调整后将通过平台首页公告、站内信等方式通知乙方，若乙方继续使用平台服务，视为同意调整后的内容。
5. 有权对乙方的主体类型及信息真实性进行核验，要求乙方补充提供相关证明材料，对信息虚假或无法核验的乙方，甲方有权暂停其平台服务。

### 6.2 乙方权利与义务
1. 有权按照本协议及平台规则使用平台服务，获取并使用合规核算的积分，对平台服务提出合理的意见和建议；境外用户有权要求甲方提供平台服务的中英文版本规则。
2. 保证提供的所有信息真实有效，对其开源平台账户信息的合法性、真实性承担全部责任，若因信息虚假导致甲方或第三方损失的，乙方应承担赔偿责任。
3. 知悉并同意本协议约定的信息采集、分析、数据使用及转让/出售相关条款，配合甲方完成必要的信息核验工作。
4. 对在使用平台过程中知悉的甲方商业秘密（包括但不限于平台规则、数据接口、运营策略、合作客户信息等）承担保密义务，不得向任何第三方泄露或擅自使用。
5. 若发现账户信息泄露、积分异常等情况，应及时通知甲方，配合甲方进行核查及处理。

## 7. 个人信息保护与授权边界
1. 本协议所述乙方个人信息包括但不限于身份信息、银行账户信息、精确联系地址、未公开的联系方式等能够单独或结合其他信息识别乙方个人信息，甲方对该等信息承担严格的保密义务。
2. 甲方对乙方个人信息的使用，仅限于本协议第三条、第四条约定的服务场景，未经乙方书面同意，不得超出该范围使用，法律法规另有规定的除外。
3. 因法律法规、司法机关或行政机关的强制性要求，甲方需要披露乙方个人信息的，甲方将在法定义务范围内予以配合，并及时将相关情况通知乙方（法律法规禁止通知的除外）。
4. 乙方有权在平台指定入口查询、更正、删除其个人信息，有权撤回本协议约定的信息采集及使用授权（但撤回授权可能导致乙方无法正常使用平台积分核算、提现等服务，由此产生的后果由乙方自行承担）。

## 8. 协议的变更与终止
1. 甲方可根据法律法规更新或平台运营需要变更本协议条款，变更后将通过平台公告方式予以公示，公示期满后即为生效，公示期不少于3个工作日，若乙方不同意变更后的条款，可停止使用平台服务并注销账户，否则视为同意。
2. 乙方可随时申请注销平台账户，账户注销前，应处理完毕未提现积分、未兑换权益等相关事宜，账户注销后，本协议自动终止，甲方将按照法律法规要求留存乙方相关信息，留存期满后予以删除或匿名化处理。
3. 乙方存在本协议约定的违规行为，甲方有权单方终止本协议，注销乙方账户，追回违规获取的积分及权益，并追究乙方的法律责任。
4. 本协议终止后，甲方对乙方的个人信息保护义务及乙方的商业秘密保密义务，不因协议终止而解除，仍应继续履行。

## 9. 争议解决
1. 甲乙双方因本协议的签订、履行、解释及争议产生的纠纷，应首先通过友好协商解决；协商不成的，任何一方均有权向甲方所在地有管辖权的人民法院提起诉讼。
2. 争议解决期间，除争议事项外，本协议其他条款继续有效，双方应继续履行相应的权利义务。

## 10. 其他
1. 本协议未尽事宜，甲方可通过平台公示的规则、公告等进行补充，补充内容与本协议具有同等法律效力；针对境外用户，需提供英文版本。
2. 本协议的任何条款被认定为无效、可撤销或不可执行，不影响其他条款的有效性及可执行性。
3. 本协议构成甲乙双方就平台注册使用事宜的完整约定，取代双方此前就该事宜达成的口头或书面约定。
4. 本协议一式两份，甲方执一份，乙方电子留存一份，具有同等法律效力。
